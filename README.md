This project is deployed on real hardware.

### Deployment Steps
1. Connect ESP32 to system via USB
2. Open the project in VS Code / Arduino-compatible environment
3. Select correct ESP32 board and COM port
4. Upload the code to ESP32
5. Insert microSD card with `001.mp3` into DFPlayer Mini
6. Power the robot using external supply

The robot runs autonomously after power-up.
